# 🧠 Personality Analyzer App

An interactive web application that determines a user's personality type based on their responses to 10 situational questions.

Built using **HTML, CSS, and Vanilla JavaScript**.

---

## 🚀 Features

- Dynamic question rendering  
- 10 personality-based questions  
- Real-time score tracking  
- Automatic result calculation (highest score logic)  
- Active button selection highlight  
- Hover effects for better UX  
- Restart functionality  
- Clean and simple UI  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3 (Flexbox + transitions)  
- JavaScript (DOM manipulation & logic handling)  

---

## 🧩 How It Works

1. User clicks **Start**  
2. Questions appear one by one  
3. Each selected option increases its corresponding personality score  
4. After the final question:  
   - The highest score is calculated  
   - The matching personality type is displayed  
5. User can restart the quiz anytime
